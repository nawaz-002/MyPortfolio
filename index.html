Simple Calculator

This project is a functional calculator that allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. Built using HTML, CSS, and JavaScript, the calculator features a user-friendly interface with clear buttons and responsive design. The goal was to create a seamless experience where users can quickly calculate results without any distractions. View Project

To-Do List App

The To-Do List App is a practical application designed to help users organize their tasks efficiently. Built with React, it allows users to add, edit, and delete tasks with ease. The app uses local storage to ensure that tasks persist even after the browser is closed. It also includes features like marking tasks as complete and filtering tasks based on their status, making it a versatile tool for personal productivity. View Project

Netflix Clone

The Netflix Clone showcases a movie streaming interface inspired by the popular platform. Built with React and Firebase, this project includes features like user authentication, dynamic content loading, and a responsive design that adapts to various screen sizes. Users can browse through a library of movies and TV shows, view details about each title, and even create a watchlist. This project was a great way to explore complex state management and real-time database integration. View Project
